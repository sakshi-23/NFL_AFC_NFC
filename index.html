Visualization Overview

This visualization presents each NFL team's drafting history. For each team, the past 10 years of draft choices are shown. Each circle is a player. Inside the circle is the position of the player. The circles are colored using that player's value for the selected mode - Player Status, Approximate Value and Games Started.
  • Clicking on the mini version of any team's draft history displays the team's history details in space below
  • For basic player information, mouse-over the circles in the detailed draft view.
  • To see complete information about a player, click on the player circle in the detailed draft view
  • For more information about the three modes, hover over the names of the modes.
